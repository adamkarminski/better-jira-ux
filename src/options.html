Setup your way to a Better Jira UX! 🚀

Go to your Jira and copy the main part of its URL, e.g. https://company.atlassian.net

An API token is a unique, secret password that the extension will use to connect to your Jira. You can create an API token here. You can name it as you like, e.g. Better Jira UX.

Default is 150, but it includes installed apps etc. If you don't see some of your users in the assign dropdown - try to increase the number.
